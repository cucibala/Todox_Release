## v3.9.0 - 2025-10-21

### 重要更新
- 数据存储迁移至 SQLite
- 自动从 JSON 迁移数据（原文件自动备份）
- 数据库性能和可靠性大幅提升
- 数据库版本管理系统（当前 v2）
- 添加任务或进度支持多照片和视频粘贴

### 新增
- 版本自动检测与迁移
- 逐版本平滑升级
- 版本历史记录
- 原生模块自动重建（postinstall）
- 子任务权重/必填输入字段支持
- 自动清理无项目归属任务

### 优化
- WAL 模式提升并发性能
- 事务处理保证数据一致性
- 索引优化查询速度

### 修复
- 自定义数据路径无法正确加载数据库
- 数据迁移时字段名兼容性问题
- 项目导入时任务ID变为浮点数
- 进度记录未正确迁移
- 子任务权重/必填输入未保存
- 打包文件缺失database.js
